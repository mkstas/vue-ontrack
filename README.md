# OnTrack

Разработано по [видео-урокам](https://www.youtube.com/watch?v=t4NbbyXsgP8&list=PL-FhWbGlJPfaCm9Qx7G9wQqtt2_yBT92V&index=2&ab_channel=ИгорьБабко)

## Установка проекта

```sh
npm install
```

### Сборка для разработки

```sh
npm run dev
```

### Сборка и минификация для продакшина

```sh
npm run build
```

### Линтер

```sh
npm run lint
```
